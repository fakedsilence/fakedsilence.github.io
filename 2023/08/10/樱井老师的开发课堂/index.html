图像设计

重视光源而非素材绘制 Draw the Light,Not the Asset

不要过分绘制素材，要表现出光影效果。 最终成品，至关重要的是应该能表达出你的素材环境里的光线反射。

突出存在碰撞判定的区域 Emphasize Objects With Collision

我们制作场地时需要重视存在碰撞体积的对象，而非强调无碰撞体积的对象。

但是使用3D多边形来制作立体视角之时，碰撞点就会变得难以分辨，也就是说，要是不把地形元素和非地形元素分开，玩家就无法得知地形具体的位置。

先突出有碰撞体积的区域，再降低玩家对背景的专注度。

要做好画面与玩法之间的平衡。 美术人员可不仅是每天做些立绘就行了的， 需要明确理解自己作品在游戏中的定位，以让它更好的融入其中。

游戏性

压力的施加与释放 Squeeze and Release

游戏如何对玩家施压，又如何对玩家解压。

在玩家完成设置的挑战后，我们应该奖励玩家，为她们提供更有效能处理压力要素的手段。

不需要去直接模仿其他游戏的压力处理方法，大家应该研究一下这些游戏为何要如此设计。

风险回报理论 Game Essnece

Risk and Reward

《太空侵略者》
风险：增加
收益：增加

风险：显著增加
收益：显著增加

这就是游戏性的本质。

游戏性是指控制风险并获得回报的乐趣。 Game Essence = The fun of managing risks/getting rewards 攻略是指控制风险并获得回报的方法。 Strategy = How you manage risks/get rewards

能控制风险的机制也是游戏平衡性的关键。

攻略：控制并降低”风险”，从而更容易获得“回报”。![QQ图片20221114213039](C:\Users\1\Documents\Obsidian Vault\图片文件夹\QQ图片20221114213039.png)

制作一款游戏时，一定要注意风险与收益。

《超级马里奥兄弟》
风险最大时，同时也是击杀绿乌龟的时间。
击杀后，绿乌龟同时拥有风险与回报。

游戏性 公众性 很大程度上是反比！

RPG的游戏性 Game Essence in Role-Playing Games

RPG游戏需要构造一个让人随时面对未知挑战的冒险，探险的区域越为深入，则亦将面对越为凶恶的敌人。
但与此同时，挑战越危险的地方得到的收益也越大。

若在一场战斗中失败，则会失去所有收益，也是风险和收益的体现。

但并不是说不会提供高难度挑战，提高特定战斗的难度，也不失为一种好平衡。

针对游戏的弱点选用攻击招式以达到输出最大化，也是一种游戏性。

连击为对战带来了一定的多样性和节奏感。

RPG游戏两个核心游戏性元素：探索、战斗。

游戏性之外的乐趣 Fun Beyond Game Essence

由操作自身带来乐趣

电子游戏可以仅通过操作来提供乐趣，若仅通过操作来提供乐趣，对创作者而言，需要选好合适的设计方式，也要有感触及调谐改善手感的能力。

冒险游戏和视觉小说

这类游戏并不重视”风险回报”，乐趣的来源侧重点在剧情推进之上。

虽然它们可以被看作是小说或者电影的延伸产物，但仅靠一个引人注目的故事本身并不足以称为优秀的游戏作品。与小说不同，游戏可以利用好电脑的交互，它们利用这一点来构建剧情，因此也成了此类游戏的乐趣以及魔力所在。

电影式以及剧情为主的游戏

这一类的剧情元素可以融入各种游戏中，无论游戏类型。
电影式的表达手法以及剧情本身并非游戏性的必要组成要素。

也可以为提升游戏性的关键，推动玩家向前直面前方危机的驱动力。

此外，鲜活的角色以及世界观有助于加深玩家对游戏的印象。

这部游戏是否可让IP的粉丝们在这个游戏世界中获得愉快的体验。如果并不是这个IP的受众，很难获得好的游戏体验，对于这样的游戏来说，游戏性并不在最主要的位置上。

生活模拟类游戏

这些游戏重现了现实生活之中错综复杂的体验。
在一个模拟的世界里玩一个拟真的游戏能提供许多游戏性以外的乐趣。

这些游戏本身仅靠模拟真实细节就已经提供了足够多的乐趣。

建造类游戏

这种游戏的乐趣本身就在于创造，但要配合具体机制。
这样就为此类游戏带来了风险回报，让游戏能提供更充足的乐趣。

建造类游戏和生活模拟类游戏存在交集。

节奏游戏

从音乐节奏得来快感乃我们的天性，而非游戏提供的乐趣。
对于这类游戏，玩家对当前曲目越为熟悉，也就越能体验到乐趣。

此类游戏围绕着”熟悉歌曲再打对节奏”进行设计。

体育类游戏

用数字化的方式重现体育，本身就存在一定的游戏性。

如果一个玩家本身就是某支队伍的粉丝，在游戏里又用的是这个队伍，就能更加强烈感受到这个游戏里的起起伏伏。

总结：游戏性并非提供游戏乐趣的唯一要素。
