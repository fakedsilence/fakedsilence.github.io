哈希表-unordered_set

关于哈希表的一种数据结构—std::unordered_set

首先，我们从上篇文章了解到，unordered_set的底层实现是哈希表，它是无序的，不可重复的，不可更改数值的，并且查找效率

和增删效率都是O(1)。

主要介绍unordered_set的几种成员方法。

1.背景

C++11为STL标准库增添了4中无序哈希容器，其中就有unordered_set容器。

unordered_set容器可直译为”无序set容器”，即unordered_set容器和set容器很像，唯一的区别就在于set容器会自行对存储的数据

进行排序，而unordered_set容器不会。

2.特性

1.不再以键值对的形式存储数据，而是直接存储数据的值。

2.容器内部存储的各个元素的值都不相等，且不能被修改。

3.不会对内部存储的数据进行排序(这和该容器底层采用哈希表结构存储数据有关)。

3.使用该容器的条件

另外，实现unordered_set容器的模板定义在头文件，并位于std命名空间中。这意味着，如果程序中需要使用该类型

容器，则首先应该包含如下代码

#include

using namespace std;

4.成员函数

1.unordered_set.find(key); 查找值为key的元素，如果找到，就返回该元素的正向迭代器。反之，则返回结束迭代器。

2.unordered_set.end(); 直接返回尾后迭代器，即指向容器最后一个数据单元 + 1的指针。

3.unordered_set result; 定义哈希表存放结果。

4.unordered_setresult(nums1.begin(),nums2.begin()); 实现功能：去除重复的元素。 [1,2,2,1] -> [1,2,1]
